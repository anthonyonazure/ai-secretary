# @aisecretary/shared

Cross-cutting zod schemas, TypeScript types, and module-output contracts shared across `apps/*` and `packages/*`. Owns request/response shapes, validated env loaders, and shared enums (regions, roles, vertical IDs). Importable from frontend and backend alike — no Node-only or DOM-only APIs allowed here.

See `CLAUDE.md` § Validation and § Coding Conventions for naming and schema discipline.
