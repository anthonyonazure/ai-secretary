What's in the box

Each item maps to source paths the GitHub repo browser can follow.

Multi-tenancy

RLS-enforced isolation

Every tenant-scoped table carries a row-level security policy. Region pinning enforced at the database layer via trigger.

AI gateways

Compliance-aware routing

HIPAA tenants → AWS Bedrock + Azure OpenAI HIPAA. EU tenants → Bedrock EU + Voyage. Per-tenant matrix.

CRM

HubSpot · Salesforce · Pipedrive

Server-side gateway, idempotent push, OAuth token rotation, Manifest V3 Chrome extension overlay.

Audit

Append-only at the SQL level

REVOKE on UPDATE/DELETE for the app role. The audit log is the source of truth for compliance evidence.

Encryption

KMS-style envelope

AES-256-GCM with rotatable Key Encryption Key. Retired KEKs stay readable; one-line swap to real KMS.

FSM gates

Tenant lifecycle + bot session

DPA + region-pin gate before any recording-pipeline mutation. Bot session FSM with watchdog heartbeats.

Visual system

Empty-state illustrations from the design brief — token-driven, no mascots.

Compliance posture

Technical controls live in code. Operational program (auditor, pen test, training) is out of portfolio scope.

HIPAA

§ 164.308 / 310 / 312 mapped

Posture-routing matrix forces BAA-covered providers; per-participant consent stricter than statute.

GDPR

Articles 5–7, 12–22, 25, 28, 30, 32–34, 35, 44–49

DSAR portal, erasure cascade walker, region pinning at DB trigger, sub-processor list with DPA + SCC.

SOC 2

Trust Services Criteria mapped

Common criteria + Availability + Confidentiality + Privacy. Each control links to its evidence artifact.

Threat model

STRIDE per surface

Trust boundaries documented. Compensating controls explicit. Reviewed on every ADR + sub-processor change.
